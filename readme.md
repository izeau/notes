## Markdown (only a subset, really)

Supported:
  * Unordered lists
  * Ordered lists (with auto-numbering)
  * Links (no reference-link yet) -- Use cmd+click to open them
  * Quotes
  * Code blocks (no syntax highlighting yet)
  * Inline code
  * Headers (hashes & underlines)
  * Horizontal rules
  * Emphasis
  * Strong emphasis
